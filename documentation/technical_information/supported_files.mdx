---
title: 'Поддерживаемые форматы аудио и видео файлов'
---

> **Важно.** При отправке файла через `multipart/form-data` часть с файлом должна содержать корректный заголовок `Content-Type`, который соответствует фактическому формату файла.

## Поддерживаемые типы MIME

### **Аудио**

- **AIFF**
  - `audio/aiff`
  - `audio/x-aiff`
- **ASF**
  - `audio/asf`
- **FLAC**
  - `audio/flac`
- **M4A / AAC**
  - `audio/aac`
  - `audio/mp4`
  - `audio/mp4a-latm`
  - `audio/mpeg4`
  - `audio/x-m4a`
- **MP3**
  - `audio/mp3`
  - `audio/mpeg`
  - `audio/mpg`
  - `audio/x-mpeg`
- **OGG (Vorbis/Opus)**
  - `audio/oga`
  - `audio/ogg`
- **Opus**
  - `audio/opus`
- **WAV**
  - `audio/wav`
  - `audio/wave`
  - `audio/x-wav`

### **Видео (извлекается только аудио)**

- **AVI**
  - `video/x-msvideo`
- **MKV**
  - `video/x-matroska`
- **MOV**
  - `video/quicktime`
- **MP4**
  - `video/mp4`
